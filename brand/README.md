# UtahLister Brand Guide

This folder holds the public-safe brand rules for UtahLister.

Private brand implementation files such as logos, exported assets, and design-system code are intentionally kept out of this repo.

## Core Brand

- Brand name: `UtahLister`
- Slogan: `Clearer listings. Better buyers. Less seller work.`
- Positioning line: `Utah marketplace listing studio for stronger Facebook Marketplace and KSL listings.`
- Trust line: `You keep the account and payout. UtahLister builds the listing package.`

## Voice Rules

Write like:

- a calm local operator
- a practical guide
- a brand that values trust over hype

Do not write like:

- a vague AI product
- a manipulative ad funnel
- a reseller coach selling status

## Proof Rules

- customer proof leads whenever available
- founder proof must be labeled `Founder example case study`
- no unlabeled personal inventory belongs in public brand work
- every proof claim should be traceable to a case-study doc or approved internal proof record

## Visual Direction

- warm stone background
- charcoal typography
- canyon red accent
- clean premium restraint
- bold but calm layouts
