# Website And Advertising Plan

This document is the canonical plan for `utahlister.com` and the first phases of promotion.

## The Website's Real Job

The site does not need to impress a designer first.

It needs to help a cautious local seller decide:

- this is real
- this is clear
- this feels safe
- I can start with one item

## Homepage Message Order

The homepage should communicate, in this order:

1. what UtahLister is
2. who it is for
3. why it is trustworthy
4. what the seller gets
5. how the process works
6. how to start

## Required Trust Stack

The live site should always make these points obvious:

- seller keeps the account and payout
- no normal password handoff
- proof is visible and specific
- founder proof is clearly labeled as example proof
- the offer is productized and easy to understand

## Current Website Scope

The site should stay simple in this phase:

- one strong homepage
- working intake form
- clear thank-you flow
- visible proof and boundaries

Nice-to-have later:

- proof library page
- sample package download
- location or category landing pages

## Advertising And Promotion Phases

### Phase 1: trust-first launch

Use:

- referrals
- partner outreach
- Google Business Profile
- KSL service listing
- local social and neighborhood channels

The goal is proof and reviews, not traffic for its own sake.

### Phase 2: proof expansion

Once customer proof exists:

- expand the proof library
- strengthen Google review flow
- create category-specific proof snippets
- improve local discoverability

### Phase 3: paid experimentation

Only test paid channels after:

- the website converts reliably
- the intake quality is strong
- the offer is stable
- proof and review volume are real

## Launch Checklist

- slogan and logo are final
- homepage copy matches the productized offer
- proof labels are correct
- unused assets are removed
- form destination is live
- thank-you page is aligned
- repo docs and site copy do not contradict each other

## Success Metrics

Use these signals to judge the site:

- intake completion rate
- percentage of leads that fit the offer
- percentage of leads that become paid work
- referral rate
- proof-worthy outcomes created from site-generated leads

If traffic rises but good intakes do not, fix the message before buying more attention.
