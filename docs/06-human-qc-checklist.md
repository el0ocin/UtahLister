# Human QC Checklist

Run this before every listing goes live.

## Claim accuracy
- brand and model are visible or clearly marked uncertain
- condition matches what the photos actually show
- included accessories are visible or were provided by the seller
- no promises about performance that cannot be verified

## Copy quality
- title is specific, not generic
- first sentence includes the strongest differentiator
- description is short enough to skim
- tone sounds like a real seller, not a chatbot
- flaws are disclosed clearly and calmly

## Local fit
- city / region is correct
- pickup language is safe
- local landmark is helpful but not over-shared
- default platforms make sense for the item

## Pricing fit
- posted price matches the selected lane
- private floor is protected if one was provided
- negotiation note is realistic

## Photo fit
- cover image shows the item clearly
- image order builds trust
- no duplicate photo clutter
- any missing trust-building shot is noted

## Delivery fit
- buyer scripts are ready
- next best move is useful
- nothing in the output requires another long tool chain

If any of these fail, revise before posting.
