# Buyer Message Bank

Keep the tone friendly, brief, and firm.

Default rule:
- do not stop at a flat statement when an easy next-step question will move the conversation forward
- keep the ask easy to answer
- reopen the path to pickup whenever possible
- stay safe: no exact address until pickup timing is locked

## Is this still available?
`Yes, it is. I am free [INSERT AVAILABILITY]. Do you have any questions about the [ITEM], or is there a time that works best for you to come check it out?`

## Lowball offer
`Thanks for the offer, but I am staying at [PRICE] on this one. If that works for you, what time would you want to come pick it up?`

## What's your lowest?
`Lowest I would do right now is [PRICE]. If that works for you, when were you wanting to come by?`

## Interested buyer ready for pickup
`Sounds good. I am near [LANDMARK / AREA]. What time were you wanting to come by?`

## Hold request
`I can only hold it briefly if you are already on the way. Otherwise it will go to whoever can pick up first. Do you want to set a time for today?`

## Ghosted follow-up
`Just checking back to see if you are still interested. I have had other interest, so if you want it, what time would you like to come by?`

## Buyer asks for address too early
`I usually share the exact address once we lock in a pickup time. What time works best for you?`

## Buyer wants delivery
`I am keeping this one as pickup only. If pickup works for you, what time were you thinking?`

## Scam / weird payment request
`I only do local pickup and payment at pickup. If you want it, we can set a pickup time.`

## Fast-close version
`If you want it today, I can do [PRICE] for pickup near [LANDMARK].`

## Price-firm version that still moves the sale
`I am already at the lowest price I want to take, so I would stay at [PRICE]. If you still want it, when would you want to come get it?`
