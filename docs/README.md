## Public-Safe Repo Rule

This docs set is meant to be safe for a public business repository.

That means the repo should explain:

- what UtahLister is
- how the service works
- where the business is in the build
- what the roadmap looks like

It should not expose:

- source code
- private tooling internals
- customer data
- raw proof screenshots
- exact deployment or security implementation details
- private app architecture notes or implementation files
