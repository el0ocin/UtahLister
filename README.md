# UtahLister

UtahLister is a Utah-focused done-for-you marketplace listing service.

The business helps local sellers turn item photos and a few basic details into a stronger Facebook Marketplace and KSL listing package with less work on the seller's side.

This repository is the public-safe master source for understanding the business. It is intentionally documentation-first and excludes source code, private tooling, customer data, raw proof screenshots, and sensitive security or deployment details.

## What Is Intentionally Excluded

- source code and automation scripts
- website or app source files
- private runtime and deployment configuration
- customer photos, raw proof screenshots, and working folders
- credentials, exact workspace configs, or private internal URLs
- raw security audits and detailed implementation findings

## Current Phase

UtahLister is in structured pre-launch as a service business.

- The offer, workflow, positioning, research base, and outreach system are defined.
- The private operator tooling exists, but it is intentionally not tracked here.
- The next milestone is repeatable customer proof, a stable intake-to-delivery flow, and a cleaner public launch posture.
