# UtahLister Public Site

This folder contains the current static site files for `utahlister.com`.

## Keep

- the current HTML, CSS, and JS used by the live page
- the favicon and any image files referenced by the live site
- the intake checklist if it is linked from the live site

## Proof Policy

- customer proof should lead whenever available
- founder proof can appear only when it is clearly labeled `Founder example case study`
- no unlabeled personal inventory belongs on the public site

## Do Not Keep

- stale preview screenshots
- duplicate asset folders
- export zips
- one-off test images not referenced by the page

## Publish Rule

Before uploading:

1. confirm `config.js` points to the correct live intake destination
2. confirm the slogan, trust line, and offer match the current UtahLister docs
3. confirm proof labels are still correct
4. confirm every referenced image exists and every unreferenced asset is removed
