# Client Intake Template

Copy and send this to each client.

```text
UtahLister Intake

Please send this for each item:

1. Item name:
2. Short notes about the item:
3. What is included:
4. Condition:
5. City / region:
6. Pickup landmark or neighborhood:
7. Sale goal: quick sale, max profit, or not sure yet:
8. Private hard floor if you already know it (optional):
9. Platforms allowed:
10. Photo folder link:
11. Preferred buyer contact:
12. Posting support preference:

Optional but helpful:
- original retail price if you know it
- why you are selling it
- anything that should be disclosed
- whether you want porch pickup, driveway pickup, or public meetup
- if you do not know the price, leave that to UtahLister
```

## Internal rule
Do not start the job until the intake is clean enough to post without guessing the important details.

## Safe sharing note
- send a view-only photo folder link when possible
- do not send your exact home address in the intake
- if a folder includes unrelated personal documents or family photos, remove those first
