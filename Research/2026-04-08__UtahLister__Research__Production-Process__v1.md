## Recommended Customer Deliverables  
Each completed job should yield a **posting-ready listing package**, including:  
- **Title/Headline:** A concise, keyword-rich title (brand, item, condition, color) optimized for search【11†L304-L312】【17†L49-L57】. For example, KSL advises using all characters (brand+model+features) to grab attention【17†L46-L54】.  
- **Description:** A structured description written clearly and honestly. It should *hook* the reader with the main benefit, then list key specs (dimensions, condition, accessories), disclose any flaws, and conclude with selling details (why selling, terms)【11†L327-L336】【18†L151-L157】.  (Use short paragraphs or bullets so buyers can scan.)  
- **Pricing Recommendation:** A **fast-sale price** (to sell quickly) and a higher **hold price** (best-case) based on local market research【11†L361-L364】【33†L112-L121】.  For example, Titan’s guide suggests setting list price ~15–20% above your minimum to allow negotiation【11†L361-L364】.  The package should also include a “floor guardrail” (absolute bottom price)【33†L112-L121】.  
- **Photo Guidance:** A **recommended photo order**, with the best cover image first and supporting shots (different angles, close-ups) following【11†L274-L277】【33†L126-L129】.  High-quality, well-lit photos are crucial (good images can triple inquiry rates【11†L270-L278】).  The deliverable should note which image to use as the first/cover photo and suggest the sequence to highlight features.  
- **Buyer Reply Scripts:** Short, friendly response prompts to common questions, steering the conversation toward a sale (e.g. confirming availability, arranging pickup)【33†L130-L136】.  UtahLister’s site highlights “question-led replies” that encourage meeting in person【33†L130-L136】.  
- **Category Recommendation:** The best marketplace category (especially on KSL). E.g. specify the subcategory path (as KSL instructs: “pick category and subcategory” for visibility)【20†L241-L249】【5†L169-L174】.  Correct categorization makes the listing searchable.  
- **Platform Recommendation:** Guidance on where to list (Facebook Marketplace, KSL, or both).  KSL and Facebook have different strengths: KSL offers a safer, Utah-focused audience【26†L154-L162】 while Facebook has broader reach.  If an item suits one platform better (e.g. furniture on KSL vs fashion on FB), note it.  UtahLister should encourage listing on both for maximum exposure, but flag the *stronger* lead (e.g. KSL for local vehicle, FB for smaller items).  

Each deliverable should come as clearly labeled fields (see “Delivery Format” below).

## Recommended Internal Drafting Process  
**1. Intake:** Gather all input (5–12 photos, item details, location, and seller’s lowest acceptable price)【33†L154-L160】. Log these in a job ticket.  
**2. Initial Research:** Search local Facebook Marketplace and KSL for comparable items (similar condition and specs) within ~25 miles【11†L351-L359】.  Note market prices and common wording/keywords.  
**3. Pricing Analysis:** Based on the comps, set a *fast-sale price* (for quick turn) and a *hold price* (slightly higher). Titan’s guide suggests putting list price ~15–20% above your floor to allow negotiation【11†L361-L364】. Record reasoning (e.g. “similar bike listed $500”).  
**4. Photo Selection:** From the submitted images, choose the strongest as the cover: an uncluttered, well-lit shot showing the item clearly【11†L274-L277】. Sequence the rest (e.g. detail shots, brand labels, wear). Follow KSL’s advice: use at least 5 angles and include any imperfections【17†L73-L81】.  
**5. Title Drafting:** Write a headline using the agreed formula (e.g. “[Brand] [Item] – [Condition] – [Key Feature/Color]”)【11†L304-L312】【17†L49-L57】. Include brand/model first, then condition (like “Excellent”), then one or two strong features. Keep it under the platform’s char limit (KSL allows ~95 chars【17†L46-L54】).  Use relevant search terms (keywords) exactly as a buyer would search【17†L49-L57】【11†L304-L312】.  
**6. Description Drafting:** Compose the description in sections: start with a **hook** that highlights the item’s main benefit, then bullet or short paragraphs for **specs** (dimensions, model numbers, included accessories), clearly note **condition** and any **defects** (KSL: be honest about wear【18†L151-L157】), and include **transaction terms** (cash-only pickup, etc.).  Mention why seller is selling if space allows (builds trust)【18†L151-L157】. Ensure keywords (material, color, usage) appear naturally【17†L135-L142】【11†L327-L330】.  
**7. Category/Platform Assignment:** Choose the KSL subcategory (e.g. Home > Furniture > Chairs) per KSL’s own guidance【20†L241-L249】. Also note the Facebook Marketplace category.  If an item is better suited to one platform (based on type and local relevance), flag that as the “first move.” UtahLister’s site suggests giving customers a “best first move” for FB or KSL【33†L163-L170】.  
**8. Buyer Replies:** Draft a few ready-made replies to likely buyer inquiries (“Is this still available?”, “Any scratches?”). Keep them short and question-led (to keep the buyer engaged)【33†L130-L136】. For example: “Yes, it’s available. How soon are you looking to pick it up?”  
**9. Assembly:** Compile everything into the deliverable format: labeled fields or sections (Title, Description, etc.). Include any notes (e.g. “Cover Image: IMG_1234.jpg”). Double-check that no fields are missing.  
**10. Self-Review (Pre-QC):** Before sending to QC, run through the pre-checklist (below) to catch obvious errors or omissions. 

Throughout, use standardized guidelines for each piece (see below) but adapt wording/tone to the item (e.g. more technical for tools, more lifestyle for decor).

## What Should Be Standardized vs Flexible  
- **Standardized:** Use a fixed **structure**. Title format (Brand+Item+Condition+KeyFeature)【11†L304-L312】【17†L49-L57】, description sections (hook, specs, condition, terms)【11†L327-L336】, and photo guidelines (minimum number and cover image)【11†L274-L277】 should be consistent.  The pricing method (market comps + 15–20% buffer)【11†L361-L364】 and inclusion of both fast and hold prices is standard【33†L112-L121】. All listings must include pickup/payment terms (e.g. “cash only – pickup in X”) in the description. The **delivery format** (e.g. a one-page PDF with labeled sections) should be uniform for every job【29†L80-L88】.  
- **Flexible:** The *content style* can vary by item type. For example, an electronics listing might use industry jargon and bullet-point specs, while a clothing or decor item might use more emotive language and lifestyle descriptors.  Include category-specific details only when needed (e.g. dimensions for furniture【18†L169-L177】, not for a small lamp).  Tone and example dialog in buyer replies can adapt to the item (formal vs casual). Platform choice may differ: luxury items might prefer KSL’s vetted community, whereas everyday goods get posted to both KSL and the seller’s Facebook Marketplace.  In short, standardize the *process* and *layout*, but allow judgment in phrasing details and emphasis based on item characteristics.

## Drafting Standards by Deliverable Type  
- **Title:** Must clearly convey *what it is* using key terms. Follow the formula ([Brand] [Type] – [Condition] – [Feature/Color])【11†L304-L312】【17†L49-L57】. Avoid vague titles like “Great bike” or ALL CAPS/SPAMMY words. Include condition (“Like New”, “Excellent”) for credibility【11†L306-L314】. Keep it concise (under platform limits; e.g. ~95 chars on KSL【17†L46-L54】).  
- **Description:** Should be honest, informative and concise. Start with a brief *lead* (the item’s prime selling point) and then cover specifics: materials, dimensions/size, model numbers, number of uses, included accessories. Use bullet points for specs/measurements【17†L99-L107】【11†L327-L330】. Clearly state any flaws or missing parts (transparency builds trust【18†L151-L157】). Highlight benefits to “the buyer” to appeal emotionally (KSL: “speak to the egocentric” benefits)【17†L117-L125】, but don’t oversell. End with terms of sale (OBO? pickup only? condition of payment).  
- **Pricing Recommendation:** Must be data-driven. Use local comparable items to justify prices【11†L351-L359】. Standardize by always providing both a *fast-sale price* and a *hold price*【33†L112-L121】 (Titan calls this “dynamic pricing”【11†L374-L383】). Check that the fast price is clearly lower to move quickly, and the hold price is realistic (about 15–20% higher)【11†L361-L364】. If relevant, note “OBO” (or “firm” if not negotiable) as KSL suggests【18†L193-L199】.  
- **Platform/Category:** Always specify where and how to list. Standardize on using KSL’s exact category path (follow KSL’s subcategory selection【20†L241-L249】). On Facebook Marketplace, use the closest category (e.g. For Sale > [subcategory]). Provide a short rationale (e.g. “Best on KSL because this category has high local demand”). Emphasize if the item fits a high-interest KSL category (cars, furniture, pets)【26†L154-L162】.  
- **Photo Order:** Standard practice is first image = attention-grabber【11†L274-L277】 (clear shot of the whole item on a neutral background, upright). Subsequent photos show alternative angles, brand logos, serial tags, and any damage【17†L73-L81】【11†L274-L277】. When writing the “photo order” section, simply list or number the images in recommended sequence.  
- **Buyer Replies:** Responses should be short, polite, and keep the conversation moving. Standardize using questions or calls-to-action (“This item is available! Would you like to schedule a pickup?”)【33†L130-L136】. Always reinforce pickup/payment terms, e.g. “Cash or Venmo at pickup.” Avoid generic replies. Use a consistent friendly tone.

## Recommended Pre-QC Checklist  
Before a draft goes to formal QC, the drafter should verify:  
- **Completeness:** Every deliverable field is filled (no blank title/description/price). Photo count is ≥5. Buyer reply section has at least a couple prompts.  
- **Accuracy:** All seller-provided facts are included correctly (model, serial, included parts). The location/region is correct. The minimum acceptable price has been used properly in setting the floor/hold price.  
- **Clarity & Style:** Title and description have no obvious spelling/grammar errors (use spellcheck). Sentences are clear and paragraphs are short.  Each paragraph or bullet makes sense on its own.  
- **Consistency:** The information is self-consistent (e.g. price in text matches the numeric field, condition stated is reflected in photos). Tone is consistent (not too slangy or vague). The format (e.g. bullet style for specs) is correct.  
- **Guideline Adherence:** Title meets format (brand, etc.)【11†L304-L312】. Description sections cover the required points (benefit, specs, condition, terms). Photos are high-resolution (no obvious blurs). If any platform-specific note (like “OBO”) is needed, it’s there.  
- **No Sensitive Data:** Ensure no personal info or payment details from the seller are exposed in the draft.

If any item fails a pre-check (e.g. missing key spec), fix it immediately before QC.

## Recommended QC Checklist  
The formal QC reviewer should systematically check each category:  
- **Title:** Is it compelling and on-brand?  Check keyword inclusion (brand, model, key feature). Ensure it is under length limits (e.g. ≤95 chars【17†L46-L54】). Look for typos or nonstandard capitalization. Does it avoid clickbait (“*Free* something” etc.) and comply with platform rules (no spammy characters)? Titan cautions to “avoid trigger words” that Facebook’s filters might flag【35†L1-L4】.  
- **Description:** Verify all important details are present. Did the drafter disclose condition honestly【18†L151-L157】? Are dimensions/specs included for large items (furniture)【18†L169-L177】? Ensure no statements promise impossible features. Look for grammar, punctuation, and readability.  Confirm it doesn’t violate any platform policy (e.g. no “FDA approved” claims if not).  
- **Pricing:** Re-calculate a quick sanity check: is the fast price at or below comparable listings? Is the hold price 15–20% higher than the fast price as planned【11†L361-L364】? If the seller’s lowest acceptable price was given, ensure it’s respected as a floor.  A note: KSL warns that a price too high “you’ll never win” and too low “you lose too”【18†L185-L193】 – QC should catch any strategy that looks unreasonable.  
- **Photos:** Look at the chosen cover image: is it eye-catching and clear? (Titan says “Visual presentation drives 87% of buying decisions”【11†L270-L274】.) Ensure the photo order list matches the actual images. Are any required shots missing (e.g. damage close-ups)? If an image has obvious problems (glare, clutter), flag it. Also confirm resolution and orientation are appropriate.  
- **Category & Platform:** Double-check the chosen category matches the item【20†L241-L249】. For KSL, make sure the path is complete. If a platform suggestion was given, verify it makes sense (e.g. KSL is indeed popular for this item type【26†L154-L162】).  
- **Buyer Replies:** Read the suggested responses for tone and usefulness. Are they courteous and relevant? Do they reinforce key info (e.g. “Yes it’s available. I can meet tomorrow afternoon, cash at pickup”). They should not contain errors.  
- **Overall Compliance:** Ensure nothing violates Facebook/KSL rules (no personal contact info outside of allowed fields, no prohibited items, etc.). Check that all seller instructions (like “pickup only”) are clear either in terms or descriptions.  
- **Consistency with Input:** Finally, make sure the listing reflects everything the seller provided. If the seller mentioned some extra detail (e.g. “has original box”), confirm it appears.  

QC should use a checklist template to mark each item pass/fail.  Any major fail (e.g. missing title or blatantly wrong price) means the draft is returned for revision.  

## Pass / Fail Delivery Readiness Standards  
A draft **passes QC** only if it meets *all* of the following criteria:  
- **Complete:** All required sections (title, description, price, photos, replies, category) are present and non-empty.  
- **Clear:** Title and description are free of errors and conform to format guidelines. Key selling points and item details are clearly communicated.  
- **Accurate:** Pricing aligns with market data and seller’s input, and all facts (model, condition, specs) are correct.  
- **Compliant:** No policy-violating language or claims, platform rules are followed (e.g. no prohibited words for Facebook【35†L1-L4】). Listing terms (pickup/cash) are stated.  
- **Effective:** The listing is compelling (fits best practices like Titan’s “3-second rule”【11†L261-L264】). For example, the cover photo is strong and title grabs attention; all QC reviewers agree it looks sale-ready.  

If any of these fail (e.g. a critical typo or a seriously unrealistic price), the listing is sent back for fixing. The target is 100% adherence – partial credit isn’t allowed for missing essentials.  

## Recommended Delivery Format  
Deliver the final listing in a **concise, easy-to-use document** (e.g. PDF or Word/Google Doc) with clear labeled sections so the seller can copy/paste directly.  For example, one guide’s “Facebook Listing Template” uses headings like **Item Name/Brand**, **Selling Price**, **Size**, **Measurements**, **Colour**, **Condition**, **Description**, and **Terms & Conditions**【29†L80-L88】. UtahLister should produce a similarly structured sheet:  

- **Title:** … (the headline)  
- **Fast-Sale Price:** $X (use this price on the listing)  
- **Hold Price:** $Y (suggested higher price)  
- **Photo Order:** (list images by filename or #; specify which to upload first)  
- **Category:** e.g. Furniture > Sofa > Sectional (for KSL)  
- **Description:** … (the full text)  
- **Payment/Pickup Terms:** e.g. “Cash only, pickup near [City]. No shipping.”  
- **Buyer Reply Samples:** (brief Q&A bullets)  

This document should be formatted simply (headings or bold labels) and be at most a page or two long. Embed or attach images separately but list their recommended sequence in the file.  An example cheat-sheet even tells sellers to “copy and paste the below into your listing”【29†L78-L88】 – UtahLister’s deliverable should work just as seamlessly.  Providing it as a PDF ensures formatting stays consistent, while sending a modifiable Doc allows easy edits.  

## Recommended Customer Posting Instructions  
Along with the listing content, include step-by-step posting instructions for both platforms:

- **For KSL Classifieds:** Advise the seller to open the KSL app or website and click “Add Listing.” **Category:** Select the subcategory path we provided【20†L241-L249】. **Title:** Paste the provided title【20†L269-L272】. **Description:** Paste our description, ensuring all the details and terms are visible (KSL advises to “disclose info upfront” about any defects【20†L275-L282】). **Photos:** Upload the images in order, with the best image first (per our guidance). **Price:** Enter the fast-sale price in the price field; if we recommended “OBO,” add that abbreviation in parentheses【18†L193-L199】. **Payment & Pickup:** In the designated fields, select “Cash on pickup” (if available) and write “Local pickup only.” **Submit:** Review and click “Post my Listing.” (Reminder: KSL suggests checking all info since it’s quick and easy【20†L241-L249】【20†L269-L272】.)

- **For Facebook Marketplace:** Instruct the seller to go to Facebook Marketplace and choose “Sell Something” (or “Items for Sale”). **Category:** Select the matching category (e.g. “Home & Garden > Furniture > Sofas”). **Title:** Paste the exact title. **Photos:** Upload images in our suggested order (Facebook allows multiple; ensure the cover is first). **Price:** Enter the fast price. If KSL’s “hold price” is higher (optional on FB), either ignore it or mention it in the description (e.g. “$X OBO”). **Description:** Paste the description text, including terms (Facebook doesn’t have a separate terms field; put “cash only, pickup” at the end). **Location:** Confirm the pickup location/zip code. **Publish:** Click “Post.” Emphasize that using the provided content verbatim (and following these steps) will make the listing look polished.  

By giving these instructions, UtahLister ensures customers know exactly how to implement the deliverable.  We draw directly from KSL’s advice (pick the right category【20†L241-L249】, write a specific title【20†L269-L272】, and disclose conditions【20†L275-L282】) to make the posting foolproof.  

## Risks and Common Fulfillment Mistakes  
QC should focus on these weak points, as they are the top causes of poor outcomes:  

- **Title/Visibility Issues:** A title that’s too generic or missing keywords will not appear in searches. Titan warns that buyers give only ~3 seconds to a listing before scrolling on【11†L261-L264】, so a boring title/image is essentially invisible. Ensure the title follows best practice formula【11†L304-L312】【17†L49-L57】.  
- **Missing Category:** Skipping or mis-selecting the correct category is a fatal error. KSL and Facebook both emphasize choosing the right category/subcategory【20†L241-L249】【5†L169-L174】, which greatly affects visibility. QC should flag any listing where category isn’t set or is nonspecific.  
- **Incomplete Details:** Forgetting to include measurements, brand/model, or the reason for selling frustrates buyers. For example, KSL’s tips stress always adding dimensions for big items【18†L169-L177】. Failing to note flaws (scratches, etc.) can lead to returns or complaints. QC must catch any missing bullet (e.g. “Dimensions: ___”).  
- **Pricing Mistakes:** Setting price too high (or fiddling with comps) often results in no sale. KSL explicitly warns “list too high and you’ll never win”【18†L185-L193】. QC should verify our pricing aligns with research. Similarly, pricing too low can betray seller’s time (some sellers prefer using the “Free” category if it’s not worth very much).  
- **Image Blunders:** Using a poor-quality cover photo (dark, blurry, cluttered) is a common pitfall. Expert sources show professional photos drive far more clicks【11†L270-L278】, so QC must ensure the first image is crisp and appealing. Also check that images do not contain personal items or extraneous objects (KSL: “nobody wants to see your dirty laundry”【17†L80-L89】).  
- **Grammar/Clarity Errors:** Typos or long-winded copy reduce credibility. A sloppy description undermines trust. QC should catch all spelling/grammar mistakes and ensure short, punchy sentences as recommended【11†L327-L336】【17†L99-L107】.  
- **Platform Rule Violations:** Using prohibited terms (like “FDA approved”) or sharing personal contact info can get listings removed. Titan warns against Facebook “trigger words” that tripping AI filters【35†L1-L4】. QC must eliminate any unsafe claims or non-compliant phrases.  
- **Altered Deliverable:** If the customer changes our text significantly (e.g. rewrites the description or omits our price), UtahLister’s metrics and guarantees break down. UtahLister should note in terms that we cannot vouch for results if the listing is modified after delivery. A manual task (post-report) should be to decide a policy for revisions or refunds if customers don’t use the deliverable as provided.  

By focusing QC on these common issues, UtahLister can maintain a high quality bar and catch mistakes early.
